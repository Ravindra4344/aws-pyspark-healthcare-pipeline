# AWS PySpark Healthcare Claims Pipeline

This project demonstrates a scalable data engineering pipeline for processing healthcare claims data.

## 🚀 Tech Stack
- AWS S3
- PySpark
- Apache Kafka
- Databricks
- SQL

## 📊 Features
- Ingest structured & semi-structured data (JSON/CSV)
- Transform data using PySpark
- Data quality checks and validation
- Build analytics-ready datasets
- Sample KPI queries for reporting

## 🏗️ Architecture
Source → S3 → PySpark → Processed Data → Analytics

## 💡 Use Case
Healthcare claims data (medical & pharmacy) processing for reporting and analytics.
